## About

I build AI products that connect language models with real workflows: agents, RAG systems, voice interfaces, automation pipelines, ML education platforms, and full-stack prototypes. My work sits at the intersection of **Deep Learning**, **Machine Learning**, **Computer Vision**, **LangGraph-style agent systems**, and production-oriented web development.

Current focus areas:

- Multi-agent applications with planning, memory, tool use, and workflow orchestration.
- Full-stack AI products using TypeScript, Python, Next.js, FastAPI, Supabase, and vector search.
- Practical GenAI systems for research, content, operations, voice, travel, invoices, and education.
- Computer vision and deep learning projects built from a strong ML foundations background.
